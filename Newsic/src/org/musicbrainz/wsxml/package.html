A parser for the Music Metadata XML Format (MMD).

This module contains {@link org.musicbrainz.wsxml.impl.JDOMParser}, which parses the Music Metadata XML Format (MMD) returned by the MusicBrainz webservice. There are also DOM helper functions in this module used by the parser which probably aren't useful to users.

You can implement your own {@link org.musicbrainz.wsxml.MbXmlParser} and inject it through one of the {@link org.musicbrainz.Query} constructors to use it.

@see org.musicbrainz.wsxml.impl.JDOMParser
